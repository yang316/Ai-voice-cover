API 接入指南

🎤 GPT-SoVITS（本地部署，效果最好）

开源免费，中文效果最好，5秒音频即可克隆。已集成到项目中。

一键安装：

  1. 安装： bash scripts/setup_gpt_sovits.sh install
  2. 下载模型：bash scripts/setup_gpt_sovits.sh download
  3. 启动服务：bash scripts/setup_gpt_sovits.sh start
  4. 在左侧设置中选择 GPT-SoVITS 后端

管理命令：

  • bash scripts/setup_gpt_sovits.sh status — 查看状态
  • bash scripts/setup_gpt_sovits.sh stop — 停止服务
  • bash scripts/setup_gpt_sovits.sh restart — 重启

训练自己的模型：

  1. 启动 WebUI：cd third_party/GPT-SoVITS && python webui.py
  2. 上传 5~10 秒参考音频
  3. 填写对应文字，选择语言
  4. 点击训练，完成后自动可用

📖 GitHub: RVC-Boss/GPT-SoVITS（58.6k ⭐）

🐟 Fish Audio（中文最强云端 API）

国内团队，中文效果好，支持声音克隆。

获取 API Key：

  1. 访问 fish.audio 注册账号
  2. 登录后进入 API Keys 页面
  3. 点击 Create API Key，复制保存
  4. 粘贴到左侧设置的 API Key 输入框

价格：

  • TTS（s2-pro）：$15 / 100万 UTF-8 字符（约12小时语音）
  • 按量付费，注册送免费额度

🔊 ElevenLabs（英文最强云端 API）

全球最流行的 TTS 服务，英文效果顶级，支持 29 种语言。

获取 API Key：

  1. 访问 elevenlabs.io 注册账号
  2. 登录后点击右上角头像 → Profile + API Key
  3. 在 API Key 区域点击 Show 复制
  4. 粘贴到左侧设置的 API Key 输入框

价格：

  • 🆓 免费：每月 10,000 字符（约10分钟）
  • Starter $6/月：30,000 字符
  • Creator $22/月：121,000 字符

🏠 本地 GPU（RVC，完全离线）

使用 RVC 模型，无需联网，需要 NVIDIA/AMD/Intel 显卡。

使用方法：

  1. 点击左侧 一键下载基础模型（首次使用）
  2. 从 voice-models.com（27,900+ 模型）或 HuggingFace 镜像站 下载人声模型
  3. 将模型文件夹放到 voices/ 目录
  4. 在音色模型下拉菜单中选择

显卡要求：

  • NVIDIA CUDA（推荐，4GB+ 显存）
  • AMD ROCm / Intel XPU / Apple MPS
  • 无显卡自动回退 CPU（较慢）
